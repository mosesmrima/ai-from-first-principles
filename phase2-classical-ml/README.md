# Phase 2 — Classical ML & Classical AI (Weeks 9–16)

> **▶ Your ordered steps live in the tracker app → https://ai-tracker.mrima.workers.dev (open "Now"). This README is reference.**

**Goal:** understand the pre-deep-learning landscape. This context makes everything later click.
Implement the core algorithms from scratch — then you'll know what the libraries actually do.

**Dependencies:** base; add `--extra nlp` only if you want `scikit-learn` for *baselines/comparison*
(never as a substitute for the from-scratch implementations).

## Resources
- Stanford CS229 — Machine Learning (Andrew Ng) — lectures + notes
- Berkeley CS188 / Stanford CS221 — Intro to AI (search, planning, value iteration)
- *An Introduction to Statistical Learning* (ISLR) — free PDF, statlearning.com
- Google ML Crash Course; StatQuest ML playlist; *Hands-On ML* (Géron)
