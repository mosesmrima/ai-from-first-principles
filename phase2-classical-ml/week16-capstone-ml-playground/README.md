# Week 16 — Capstone: ML Playground

**The Phase 2 capstone.** Bring together everything from weeks 9–15 into one small project that
does **classification, clustering, and regression** — all using your own from-scratch
implementations, on a dataset you choose (Kaggle tabular data is ideal).

**Suggested shape:**
- A clean interface to run each algorithm on a chosen dataset.
- Train/test split + cross-validation you wrote.
- Metrics you compute yourself (accuracy, MSE, silhouette/inertia for clustering).
- A short writeup comparing the models — this is portfolio-worthy.

**Checkpoint:** you can pick a dataset and reason about which of your algorithms fits the task
and why. Mark done:
```bash
uv run python track.py done week16
```
