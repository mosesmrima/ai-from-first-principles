# Phase 9 — Research Apprenticeship (Weeks 77+, ongoing)

**Goal:** become someone who reads, reproduces, and writes. This is a *practice*, not a phase
you finish. Start the weekly habit during Phase 5 and never stop.

## The cadence

- **Weekly:** read 1 paper, write a one-page summary in `../papers/` (use the template).
- **Monthly:** reproduce one result from a paper you read. Publish the code.
- **Quarterly:** publish a blog post / GitHub project / technical writeup.

## Graduation milestone
A **public GitHub repo + writeup reproducing or improving a real paper.** This is portfolio
project #10 and the thing that proves you've crossed from practitioner to researcher.
Mark it with `track.py done research-graduation`.

## Suggested reading order
GPT-1 → GPT-2 → GPT-3 → BERT → InstructGPT → Chinchilla → LLaMA → FlashAttention →
Toolformer → ReAct → Constitutional AI. (Full list + how-to-read in `../papers/README.md`.)
